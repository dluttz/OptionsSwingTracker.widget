See the workflow in 30 seconds.

Add a ticker, type the catalyst, choose the reaction date, and keep a small desktop read on the priced move and data quality.

What it answers

How much move is priced into the options chain around this event date?

How it calculates

It preserves the ATM straddle expected move math from the original Python script, including raw and 0.85 adjusted reads.

Why it is careful

Rows show quote basis, stale data, placeholder IV, padded expiries, and unreliable odds instead of hiding uncertainty.

Built for fast catalyst checks

  • Editable watchlist from the widget itself.
  • Manual catalyst date entry for earnings, legal events, product launches, and binary events.
  • Live mid, prior close, indicative, cached, and stale quote states.
  • Optional risk-neutral below/above odds for a price threshold.
  • Minimal and details modes, themes, and movable/resizable panel controls.
  • Local venv setup, tests, parity checks, and a doctor command.

Install

cd "$HOME/Desktop"
git clone https://github.com/dluttz/OptionsSwingTracker.widget.git
cd "$HOME/Desktop/OptionsSwingTracker.widget"
./bin/setup.sh

Requires macOS, Ubersicht, Python 3, and internet access for Yahoo/yfinance data. Informational only. Not financial advice.
